# Online-Learning-Engagement-Dropout-Risk-Analysis
# Overview
This project analyzes student behavior patterns on online learning platforms to identify key factors associated with dropout risk. The goal is to understand what engagement looks like — and where it breaks down.

# Key Questions
- Does engagement frequency reduce dropout risk?
- Does device type (mobile vs desktop) influence dropout?
- Does course category affect dropout rates?
- Which engagement metrics are most strongly associated with dropout risk?

# Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

# Dataset
Kaggle – Online Learning Engagement and Dropout Risk Dataset

# Key Insights
- Higher login frequency is associated with lower dropout risk
- Device type shows notable differences in engagement patterns
- Some course categories carry significantly higher dropout rates
- Login frequency and time spent are among the strongest engagement indicators
